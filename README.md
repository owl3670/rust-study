# rust-study

Rust 를 학습하기 위한 repository

https://doc.rust-lang.org/book/

위의 공식 문서를 참고하여 예제들을 따라 작성하거나 궁금한 내용을 테스트 하면서 학습을 진행.

실행을 위해 폴더로 이동


